# gestaoDeAlunos

P2 Gestão de Alunos e Cursos
Este projeto é um Sistema Web de Gestão de Alunos e Cursos desenvolvido em Java com o framework Spring Boot. Ele consolida os conceitos de Orientação a Objetos (POO), arquitetura MVC e persistência de dados utilizando JPA e


Java - Linguagem de programação principal.
Spring Boot	- Framework para estruturar a aplicação, simplificando a configuração e deployment.
Padrão MVC	- Padrão arquitetural para separação de responsabilidades.
Spring Data JPA	- Abstração para persistência de dados no banco, via Repositories.
H2 Database	- Banco de dados em memória utilizado para desenvolvimento e testes.
Thymeleaf	- Motor de templates para renderizar as páginas HTML (Camada View).
Spring Validation - Utilizado para validar dados de entrada nos formulários (Ex: e-mail, não nulo).
